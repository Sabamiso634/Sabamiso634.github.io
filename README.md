# Research demo video site

This directory is a static GitHub Pages site for sharing demo videos while the paper is still in preparation.

## Included

- `index.html` and `styles.css`
- bilingual English/Japanese language toggle with `?lang=en` and `?lang=ja`
- demo videos under `assets/`

## Before publishing

Before making the repository or GitHub Pages deployment public, ensure that unpublished paper details, datasets, figures, PDFs, notes, logs, and generated archives are not included in the published output.
